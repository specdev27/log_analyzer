****What is a Parser?****
A parser takes data in format (raw and difficult to process) and converts it into a structure, sorted and labeled, so that the program can work with it easily.

1. Takes: reads the raw data
2. Filter: If a line or data doesn't comply with the format, rotes.mdeturns NONE
3. Sort and Label: Each piece of data is sorted and receives a label with a name (example: ip, status, date, etc.)

In other words, a parser is a translator, from loose text to sorted data.
