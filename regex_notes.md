#re.match(pattern, string)
Looks for any match in a string, for example, we want to look for an "G" | Example: re.match('G', "Generation of Coders")

#re.findall(pattern, string)
Find and print all the matches in the text | Example: re.findall('Generation', 'Generation of Coders')
Output: ['Generation']
